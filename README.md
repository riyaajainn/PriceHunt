# 📱 Price Hunt - Mobile Comparison Tool

**Price Hunt** is a web-based data categorization tool that allows users to explore and compare mobile devices across different brands and specifications. By leveraging local JSON and CSV data, the app provides a fast, interactive experience for finding the right smartphone.

## 🚀 Live Demo
Check out the live project here: [https://price-hunt-sigma.vercel.app](https://price-hunt-sigma.vercel.app)

## ✨ Features
* **Smart Categorization:** Automatically organizes mobile data by brand and model.
* **Dynamic Search:** Quick search functionality to find specific devices.
* **Filter System:** Filter by RAM, Storage, and Price ranges.
* **Data-Driven:** Powered by a structured JSON dataset and CSV store mappings.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Data Format:** JSON (Mobile Specs), CSV (Store IDs)
* **Deployment:** Vercel
* **Version Control:** Git & GitHub
